# [Eco Archer Team RFID Database Attendance Tracker]

The Eco Archer RFID Attendance Tracker automates time-in/out by scanning RFID-enabled school ID cards. Logs are saved in a Google Sheet, and real-time updates are sent to the teamâ€™s Discord server. Unauthorized scans are reported.

## System Overview

- **Hardware:** ESP32, MFRC522 RFID Reader, SSD1306 OLED Display, Buzzer
- **Backend:** Google Apps Script + Google Sheets
- **IDE:** PlatformIO IDE
- **Communication:** HTTPS API calls
- **Notifications:** Discord Webhook Integration
